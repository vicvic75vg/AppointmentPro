Application Name: ApptPro

Purpose: The purpose of this application is to enable users with accounts to log in and create,
update, and delete appointments for customers in the database. This application also enables users to
create, update, and delete customers as needed. Three reports are generated based on appointments 
and contacts.

Author: Victor M Garcia Jr

Contact Information: vgarc72@wgu.edu

Application Version: 1.0.0

Date: January 28, 2022

IDE: IntelliJ IDEA 2021.3.1 (Community Edition)

SDK: AdoptOpen JDK 11.0.11.9

JavaFX Version: 11.0.2+1

Running the program: After extracting all files in the provided zip file,
open the "apptpro" folder in IntelliJ IDE. Navigate to "apptpro/apptpro/src/main/java/com/apptpro/apptpro"
and run the "Main.java" file in the IDE by right-clicking on the file and selecting "Run Main.Main()" from
the drop-down menu. 

The function of the custom report: The custom report provided in the "Contact Metrics" view populates the 
table with contacts with the most appointments assigned to them. This table displays contacts in descending order.

MySQL Connector Version 8.0.27
